My Design

For this weeks assignment, I designed a simple circuit board for XIAO RP2040 module using KiCad. For this design, I used Fablab’s KiCAD library with surface-mount parts. I started my design by adding Generic Socket Xiao module from the symbol library. This particular symbol utilizes surface-mount pin sockets as its footprint, which makes it possible to swap between different XIAO modules. I added different power symbols (3.3V and ground) to the schematics, and attached power flags to them. I then continued by adding a circuit for LED to one of the GPIO pins, which consists of an orange LED, 100ohm resistor and a ground symbol. For another GPIO, I added a circuit for a button with 10k pull-up resistor. I added an option for I2C and UART connections, using 4-pin pinsockets for both of them. For the remaining GPIO pins, I used a 5-pin pinsocket. I also added the drill bits at this phase, since they also have footprints in the PCB editor.

After passing the Electrical Rules Checker in the schematic editor, I moved on to the PCB editor. I updated the PCB from schematics and moved the individual parts around trying to find a layout with the least amount of crossing rat’s nests (the dim lines showing connections between parts). Before routing any tracks, I defined the net classes. Net classes are used for setting different rules for the tracks, such as track width and minimum clearance between tracks. For my purpose, I set the values for clearance, track width, via size and hole size, since these values have to correspond to the drill bits sizes used at the Fablab. KiCad’s PCB editor organizes different kinds of PCB information into layers. After setting the net classes, I drew the tracks connections on the Front Copper layer, which contains all copper features of the front side of the PCB. After drawing the all the connections (besides the ground), I drew a temporary board shape on the Edge Cuts layer, which defines the outline of the board. I then filled the rest of the board’s front face with copper using it as a ground. This simplifies the process of drawing the grounds, and acts as a shielding for the circuit. As an additional step, I imported the final outline of a public domain wolf I had prepared in Inkscape. Finally, I added a name and version number of the board as a text using general-purpose User Layer 1.

After passing the Design Rules Checker, I exported the final design as Gerber files for PCB milling process at the Fablab. In KiCad’s Plot window, I used Gerber as plot format, and included F.Cu and User.1 layers to the export. To help the layer alignment process later in CopperCam, I included Edge.Cuts on all layers. After plotting the initial layers, I generated drill files using the Gerber X2 format. As a final step, I used KiCad’s Gerber Viewer to check the files had formatted as intented.
